## ESLint Config

### Installation

#### Basic installation

```bash
bun add -d @yelaiii/eslint eslint
```

#### React installation

```bash
bun add -d @yelaiii/eslint eslint eslint-plugin-jsx-a11y
```

Antfu's config will install missing deps
